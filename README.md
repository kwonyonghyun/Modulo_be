# 📝 MODULO

> 모듈을 조합해서 나만의 이력서를 만들어보세요! ✨  
> 🔗 [Modulo 웹사이트 바로가기](https://modulo.co.kr/)

## 💡 프로젝트 소개

이력서 작성, 더이상 어렵지 않아요! MODULO는 여러분의 이력서 작성을 도와줄 특별한 도구입니다.
원하는 모듈을 선택하고 내용을 채우기만 하면 멋진 이력서가 완성됩니다.

## ✨ 이런 기능들이 있어요

### 🎯 모듈 조합으로 만드는 이력서
- 필요한 모듈만 쏙쏙 골라서 사용
- 드래그 앤 드롭으로 순서 변경
- 실시간으로 확인하는 미리보기
- 마음대로 커스터마이징

### 📚 다양한 모듈 제공
- 기본 정보 모듈 👤
- 학력 정보 모듈 🎓
- 경력 사항 모듈 💼
- 프로젝트 모듈 🚀
- 기술 스택 모듈 💻
- 자기소개서 모듈 ✍️

### 🎨 꾸미기 기능
- 레이아웃 조정

### 💾 편리한 저장과 관리
- PDF로 내보내기
- 여러 버전의 이력서 저장

## 🛠 사용한 기술

- Frontend: React.js
- Backend: Spring Boot, Redis, Docker
- Database: MySQL
- 배포: AWS

## 📱 이렇게 사용해보세요

1. 모듈을 선택합니다
2. 내용을 입력합니다
3. 원하는 대로 꾸며보세요
4. PDF로 저장하면 끝!
